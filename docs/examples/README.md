# Code Examples

This directory contains code examples referenced by cursor rules. Examples are organized by topic and extracted from rule files to keep rules concise and maintainable.

## Available Examples

- **testing-pglite-setup.ts** - PGLite testing setup patterns
- **tenant-isolation-test.ts** - Tenant isolation testing patterns
- **provisioning-performance-test.ts** - Performance testing for provisioning
- **provisioning-workflow.ts** - Tenant provisioning workflow and rollback
- **cloud-run-deployment.sh** - Cloud Run deployment commands
- **cloud-run-tenant-deployment.ts** - Programmatic tenant instance deployment
- **cloudflare-api-integration.ts** - Cloudflare for SaaS API integration
- **database-provisioning.ts** - Neon database provisioning patterns
- **secret-management.ts** - GCP Secret Manager patterns
- **hostname-routing.ts** - Hostname-based tenant routing patterns
- **domain-patterns.ts** - Domain-driven design patterns (Repository, Service, Route)
- **logger-usage.ts** - Structured logging patterns

## Usage

Rule files reference these examples instead of embedding code. This keeps rules concise and examples maintainable in one place.
